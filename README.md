# 面世物流 · 智能运输管理系统

## 项目简介

这是重庆面世物流有限公司的智能运输管理系统前端页面，嵌入了运输管理平台，提供运单查询、轨迹追踪、电子回单等功能。

## 功能特点

- 🚚 **运单查询**：实时查询货物运输状态
- 📍 **轨迹追踪**：GPS实时追踪货物位置
- 📄 **电子回单**：在线查看和下载电子回单
- 📱 **响应式设计**：完美适配手机、平板、电脑

## 技术栈

- HTML5 + CSS3
- iframe 嵌入秒悟应用
- CSS 裁剪法隐藏第三方品牌标识
